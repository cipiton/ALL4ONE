【项目总纲：〈天使背叛？反手与魅魔签订契约！〉短剧改编母本】

——供「AI短剧工业化改编策划」技能调用的主控大纲——

--------------------------------
一、总体改编定位
--------------------------------

1. 核心类型与卖点  
- 类型：都市奇幻 + 御兽爽文 + 校园/军方试炼 + 深渊悬疑  
- 卖点：
  - 开局全民仪式当众召唤史诗光明天使却立刻背叛，主角被公开羞辱→极强情绪钩子
  - 反向选择被诅咒魅魔「莉莉丝」，从垃圾仓库里捡神话起飞，背德感+爽点并存
  - 御兽面板+养成流：资质、等级、技能清晰可视，适合HUD和口播快速呈现
  - 多条试炼线（新手秘境、群巢迷宫、沙海副本、鬼城、万龙冢、试炼之塔）提供稳定动作场面
  - 政治/势力博弈：市政、国卫局、帝都大学、军方镇关军、深渊教团、外域黑市多方角力
  - 情绪内核：被亲人出卖（父母案真相）、天才被围猎、主角从市级小人物一路进入镇渊司核心

2. 世界规则（短剧化关键信息）
- 御兽契约体系：
  - 资质：普通 < 精英 < 英雄 < 史诗 < 传说 < 神话（后期有更高，但核心到神话）  
  - 等级：凡级 → 灵级 → 玄级 → 天级 → 君主（短剧阶段重点：凡/灵/玄/天）  
  - 御兽与御主有精神契约，可读面板（主角金手指加强版）
- 面板呈现：简化为 HUD +主角低声自语或内心旁白，避免长篇数据  
- 副本/秘境：
  - 时间流速与现实不同（一般内长外短），需在剧中用字幕/台词点明避免时间混乱
  - 积分系统决定高考/升学/资源  
- 势力层级：
  - 城市（离阳市）：市长+城卫局  
  - 帝都：帝都大学、国卫局、镇渊司、镇关军  
  - 黑暗阵营：深渊教团、外域（霓域）黑市势力、唐家/地方世家  
- 敏感术语替换（必须统一）：
  - 霓虹/某国 → 外域（或“霓域”）  
  - 掠夺者 → 夺契环（禁术道具）  
  - 地狱 → 地狱位面碎片/远古位面残片  

3. 契约与力量系统关键设定
- 余晓金手指：神级鉴定面板
  - 能看：御兽资质、等级、潜力、技能、进化路线、材料需求
- 莉莉丝契约形态（必须全剧统一表达）：
  - 「亲密·奴隶契约」= 绝对忠诚+强依赖，但保留部分独立意志与原罪权柄  
- 等级/资质统一用语：
  - 画面/台词统一形态：  
    -【资质：传说九阶 / 神话一阶】  
    -【等级：凡级九星 / 灵级五星 / 灵级九星 / 玄级 / 天级】  
- 高阶技能视觉注意：
  - 「原罪·神欲」：全属性+400%基值，有明显负担与冷却（战后需要倦怠/伤势表现）
  - 原罪领域、王权领域等需要明显的领域化视觉（色调变化+空间扭曲）

--------------------------------
二、主要人物母本
--------------------------------

1. 余晓（主角）
- 身份：离阳市普通考生→帝都大学S级学员→院长亲传→镇渊司客卿→天级一星  
- 特质：冷静理性、战略感强、善用规则，精神力异常高（穿越者底子，可简化为“精神天赋异禀”）  
- 核心能力：
  - 金手指：鉴定面板（HUD信息源）
  - 天赋1：精神力/御兽之墟扩展  
  - 天赋2：归一（后期解锁）——可将多种法则/领域熔炼成一体，高消耗、高风险  
  - 最终领域：万象道域（由原罪、王权、均衡三大法则熔炼）  
- 情感/动机线：
  - 初始：被公开羞辱、契约被夺，想活下去+报复唐家  
  - 中段：唐家与外域阴谋、父母被献祭真相曝光，仇恨与责任并行  
  - 后段：被深渊教团与帝都地下封印牵引，逐渐站到压制深渊的国家层面  
- 关键人际：
  - 莉莉丝：最核心战力+情感投射对象
  - 余婉君：姐姐，帝都S级学员+镇渊司疗愈师，提供情感支点与资源
  - 陆元展：国卫局部长+帝都大学教授，赏识+利用并存
  - 陈道山/院长：真正的师尊与精神导航者
  - 圣者林书韵：高位护法，给予玉牌与镇渊司通道

2. 莉莉丝（魅魔 / 原罪魔王线）
- 初始：史诗九阶资质被诅咒→面板降为“普通废物”，被唐家当垃圾处理  
- 转折：余晓用三千万买下，在地下室破诅→露出传说级资质  
- 晋升路线：
  - 传说一阶（夜之魔女·诅咒解除）  
  - 地狱碎片副本：血脉再蜕变，开启「原罪·神欲」，资质提升到传说九阶，等级至灵级五星  
  - 后续：完成神话级蜕变（魔王形态），原罪领域开启  
- 角色标签：
  - 表面：低沉魅惑、娇媚调笑  
  - 实质：战时极冷酷，保护主人至上，有强烈占有欲与主导欲  
- 技能风格：
  - 苦痛荆棘、死欲之雾、赦罪之镰、七罪之血 → 血雾、锁链、镰刀、荆棘视觉主色为暗紫/深红
  - 原罪·神欲：粉紫+黑金，属性倍增+敌人精神溃散
  - 原罪领域：空间挤压+情欲/绝望幻境，BOSS级场面必用  
- 与主角关系演变：
  - 从“被买下的奴隶”→“救赎者+主君”→“法则并列的伙伴”  
  - 改编时用：救主牺牲、主动献身战斗、在地狱碎片副本中反噬贪罪者等名场面强化其核心地位

3. 红玉（红发少女型御兽 / 黄泉龙鬼王线）
- 起点：在对抗赛与后续龙血池中孵化的御兽
- 路线：
  - 快速成长至灵级九星
  - 鬼城篇：黄泉引触发→发现自己与无相鬼王血脉相连  
  - 吸收鬼王之心→王权加冕，资质跃迁至神话三阶，拉起「黄泉鬼国」领域  
  - 万龙冢：吸收祖龙之血→神话九阶，成为真正的祖龙/鬼王混血  
- 人格：高傲、护短，对莉莉丝略带竞争意识，对主人忠诚但更偏“王的对等盟约”  
- 功能：  
  - 中后段主战力之一  
  - 黄泉线索与鬼城篇、父母真相的关键导引者  

4. 莫甘娜（堕天使）
- 登场：天宝阁拍卖中的“残缺藏品”，双眼被刺瞎、背负神圣枷锁  
- 发展：
  - 静室契约时与莉莉丝原罪权柄冲突，引发领域对撞名场面  
  - 被逐步解锁枷锁，翅膀从四翼→六翼→神话级堕天使  
- 角色功能：
  - 「均衡」法则代表，与莉莉丝（原罪·欲）、红玉（王权·死/龙）形成三角  
  - 象征救赎与秩序，与莉莉丝的欲望倾向形成戏剧对比  

5. 关键势力人物（用于政治/推进线）
- 肖严：离阳市市长——早期保护伞，代表地方官方  
- 林冬：城卫局局长——唐家事件执行人  
- 陆元展：国卫局部长 / 帝都大学教授——多次邀请/试探余晓，将其推向更高战场  
- 夏侯渊：镇关军少将——B-7与地狱碎片副本负责人，提供「不限品级材料」筹码  
- 圣者林书韵：高位圣者 & 镇渊司代表——鬼城后出场，赐玉牌+封口令  
- 余婉君：姐姐，帝都大学S级天才+镇渊司疗愈师  
- 陈道山 / 学院院长：收余晓为亲传，指点融合三法则，护法突破天级  
- 唐桑：唐家少爷，被夺契环事件逼疯  
- 唐珍：唐家董事长，与外域/深渊勾连的关键中层棋子  
- 暮佑集：外域“大天狗自卫队队长”，短剧可在前中期作为典型外域刺客BOSS  
- 深渊教团执事「画师」：中前期主要反派之一，在废弃歌剧院施祭的BOSS战  

--------------------------------
三、主线时间轴 & 大阶段划分
--------------------------------

【阶段1：起爆篇（城市·高考·唐家线）】  
覆盖原文约1–50章 → 适合做短剧第1季前若干集

- 主轴：  
  - 全民召唤仪式 → 当众背叛 → 地下室买魅魔 → 新手秘境逆袭 → 唐家阴谋暴露 → 帝都邀请  
- 关键节点：
  1. 人民广场全民召唤仪式  
     - 余晓召唤史诗光明天使“小雪”  
     - 当众被教会/监考官祝贺→天使却背叛契约投靠唐桑  
     - 余晓成为笑柄，唐桑耀武扬威  
  2. 初升商会地下室  
     - 唐家将被诅咒魅魔莉莉丝当废物处理  
     - 余晓用三千万买下（赌桌戏+心理战），触发金手指鉴定  
     - 破除诅咒→莉莉丝真身亮相（传说资质、魅魔觉醒）  
  3. 新手秘境高考试炼  
     - 累积分数、碾压同龄人，一路登顶  
     - 终榜公布：S级潜力、积分41350打破离阳市记录  
     - 市长肖严和城卫局林冬注意到他  
  4. 群巢迷宫（脑魔BOSS战）  
     - 精神压制场景+莉莉丝荆棘猎杀  
     - 余晓以清心玉/精神力抗住脑魔幻境  
     - 击杀BOSS获得「灵魂晶核」→开启御兽之墟  
  5. 唐家/夺契环事件  
     - 唐桑使用外域违禁品「夺契环」欲强夺莉莉丝  
     - 仪器反噬+证据链曝光→唐桑精神崩溃被捕  
     - 研究员自杀，线索中断，唐珍转入更黑暗势力  
  6. 沙海副本 & 噩梦沙蛛战  
     - 野外沙海环境+炎石巨像战斗（层岩之心掉落）  
     - 噩梦沙蛛（史诗变种）→掉落「梦魇晶壳」（可替换梦欲结晶）  
     - 为莉莉丝后续升阶埋下材料  

【阶段2：军方介入 & 地狱碎片篇】  
覆盖原文约51–100章

- 主轴：  
  - 外域刺杀→ 帝都SSS特招 → B-7军方任务 → 试炼之塔异常协议 → 地狱碎片副本 → 莉莉丝原罪觉醒 → 巨蛋  
- 关键节点：
  1. 外域刺杀失败 & 暮佑集伏笔  
     - 大天狗队长暮佑集出场，失败后被捕/讯问  
     - 暴露部分外域与唐家勾结线索  
  2. 帝都大学SSS特招 &「炎黄之星」勋章  
     - 陆元展、市长联手为余晓颁奖  
     - 帝都免试+苛刻条件：十天内突破灵级  
  3. 帝都竞技场连胜  
     - 对战多位天骄，表现精细指挥与超规格战力  
     - 获得SSS级队友邀约，为后续组团试炼埋伏笔  
  4. 军方B-7秘境任务  
     - 熔火暴君战：团队配合，突出余晓指挥智慧  
     - 截获赤流金、炽心之晶，为后期锻造&资源线服务  
  5. 试炼之塔61层 & 异常协议  
     - 塔内精神冲击与空间扭曲视觉  
     - 登顶后触发与地狱位面碎片能量共振 → 国卫/军方红色警报  
  6. 地狱位面碎片副本  
     - 黑莲池、祭坛、贪罪者、噬魂血蝠  
     - 莉莉丝以色欲原罪反制贪罪者→强化自身原罪血脉  
     - 触发「原罪·神欲」技能，全属性+400%暴杀史诗BOSS噬魂血蝠  
     - 池底发现蓝晶洞天与「传说九阶巨蛋」→ 带出交军方保管/合作争议  

【阶段3：帝都新生赛 & 深渊教团前线】  
覆盖原文约101–150章

- 主轴：  
  - 新生对抗赛→ 红玉孵化飞升 → 龙血池→ 始祖龙鳞 → 深渊教团“画师”事件 → 黄泉引线索  
- 关键节点：
  1. 军方体检 & 新生对抗赛（海选→32强→8强→决赛）  
     - 一系列单挑/团战短剧化名场面，展示各系御兽  
     - 莉莉丝展现更深层原罪力量，打出“魔王”气场  
  2. 红玉孵化  
     - 以蛋孵化的方式登场，逐战成长→灵级九星  
     - 个性鲜明的台词作为轻松调剂  
  3. 龙血池秘境  
     - 高能环境+血脉觉醒幻觉  
     - 余晓获得「始祖龙鳞」「古龙的加护（残）」  
  4. 深渊教团诱饵计划  
     - 陆元展建议用余晓引蛇出洞  
     - 废弃歌剧院祭祀：执事「画师」用活人/御兽作为画料  
     - 高能BOSS战：莉莉丝原罪领域+红玉支援，斩杀画师  
  5. 天幕山脉 / 龙陨峡谷实战  
     - 连续杂战+小Boss战，红玉级别连升  
     - 获得「黄泉引」线索  
  6. S级宝库访问权  
     - 余晓挑选「鬼王之牙」「澄心极玉」等物，埋伏鬼城篇与精神突破  

【阶段4：鬼城篇 & 家族真相】  
覆盖原文约151–200章

- 主轴：  
  - 鬼城潜入→ 黄泉之河被污染 → 深渊之心与无相鬼王 → 红玉王权加冕 → 镇渊司介入 → 父母真相 → 昊天塔夺魁 → 堕天使入场  
- 关键节点：
  1. 夜潜鬼城，遭国卫拦截  
     - 阻拦冲突→临时许可：授予S级特别行动许可  
  2. 鬼城探险  
     - 黄泉之河污染：画面为黑紫河流，怨魂翻腾  
     - 红玉血脉被冲击，痛苦抽搐  
  3. 深渊之心 & 无相鬼王  
     - 发现被深渊魔影寄生的鬼王心脏  
     - 红玉道标触发，前世记忆/血脉觉醒  
     - 莉莉丝+余晓协力净化深渊魔影，BOSS战视觉：鬼雾+裂口心脏+深渊触手  
     - 红玉吸收心脏→「王权加冕」（神话三阶）+黄泉鬼国领域雏形  
  4. 圣者林书韵出场  
     - 一击抹除残留深渊，颁布封口令  
     - 赐白玉牌+镇渊司客卿身份伏笔  
  5. 玉城第7号裂缝 & 父母案  
     - 调查旧案：林婉出卖父母→深渊献祭  
     - 情绪爆点：余晓得知真相，黑暗情绪+复仇意志强化  
  6. 昊天塔资格战 & 登顶  
     - 高强度擂台战，展示余晓已达玄级六星战力  
     - 以绝对碾压夺魁  
  7. 天宝阁拍卖会  
     - 炒作紧张竞价，余晓以高价拿下「三宝琉璃玉」与「堕天使」  
     - 顺带获得「万龙冢星图」线索，为下一阶段铺路  

【阶段5：万龙冢 & 归一·万象道域】  
覆盖原文约201–259章

- 主轴：  
  - 堕天使契约与枷锁冲突→ 千机回廊试炼→ 院长收徒与S级权限 → 东云岛升神仪式阻断 → 万龙冢试炼 → 归一天赋觉醒 → 申屠家归附 → 三法则融合 → 万象道域 → 镇渊司正式登场  
- 关键节点：
  1. 静室契约 & 神圣枷锁对撞  
     - 余晓尝试与堕天使契约  
     - 莉莉丝原罪权柄感知天使气息本能敌对  
     - 原罪领域 vs 神圣秩序 → 室内法阵爆裂，视觉高潮一  
  2. 御兽之墟三相共鸣  
     - 莉莉丝（原罪）、红玉（王权）、莫甘娜（神圣/均衡）在御兽之墟共鸣  
     - 遗留材料被自动重组吸收，强化三兽  
  3. 千机回廊 & 典狱长战  
     - 高难副本：机关、幻阵、终极典狱长Boss  
     - 余晓以指挥+技能拆解机关，突出智力流  
     - 通关后被院长正式收为亲传，开启S级权限  
  4. 东云岛升神仪式阻断  
     - 邪教/深渊企图扶持某人升神  
     - 余晓带队打断仪式，将污染净化

六、改编统一规则 & 注意事项
--------------------------------

1. 契约状态统一  
- 全程将莉莉丝对余晓的契约描述为「亲密·奴隶契约」  
- 表现方式：
  - 她会称呼“主人”，表面顺从  
  - 关键战斗中会主动违拗命令以保主人的命，体现独立权柄  
  - 对其他雌性御兽/女性角色有强烈占有欲与吃醋戏  

2. 等级/资质显示统一  
- 每当需要展示面板时，只保留：
  - 名称 + 资质 + 等级 + 1~2核心技能/天赋  
- 例：  
  -【夜之魔女·莉莉丝】  
   资质：神话一阶（原传说九阶）  
   等级：灵级九星  
   天赋：原罪·神欲 / 原罪领域  

3. 高阶技能的代价表现  
- 原罪·神欲、王权领域、万象道域使用后必须配合：
  - 余晓与御兽出现力竭/伤势/精神恍惚的镜头  
  - 对话中提及冷却或负担（“这次，至少三天不能再开原罪。”）  

4. 时间线清晰化  
- 每个大型副本开始/结束用字幕标注：
  -「副本内时间：3天 / 外界时间：6小时」  
- 防止观众对“怎么刚出来又比赛？”产生混乱  

5. 势力关系简化可视  
- 对主要势力使用固定LOGO或色彩：
  - 城卫/市政：蓝白  
  - 国卫局/帝都大学：金白  
  - 镇渊司：黑金  
  - 深渊教团：紫黑  
  - 外域势力：血红带异纹  
- 帮助短剧高速识别阵营
