执行步骤：单集名场面绑定（基于已归纳母本与先前分级／五阶段节奏规划，60集竖屏短剧）

1. 单集绑定原则
- S级优先落位：S级名场面必须占据开篇（Ep1）、阶段付费点（Ep30、Ep48）与终局反转位（Ep59）；每一 S 集做为独立高潮位并留 15–30s 专门用于社媒切片/付费预览。  
- 每集必须包含三选一：冲突 / 反转 / 情绪峰值；结尾必设强钩子（悬念 / 爽点预告 / 伏笔）。  
- “改字三元素”（改名前一句话功能 → 单字闪替特写 → 改后即时效果）与“代价镜头”为所有含改字动作集的必备镜头节点。S/A 集务必完整呈现三元素。  
- 过渡集（制作标注为“过渡”）总量不超过总集数的10%（≤6 集）；过渡集仍需包含明确小冲突或信息拆解以维持钩子密度。  
- 节奏控制：将重复升级/刷怪流程以蒙太奇与战报剪辑呈现，保证每 4–6 集至少有一集设次级高潮（A/B）。  
- 人物功能落点：为主要角色（姜延、姚可欣、王清枫、姚力、翟语晗、罗天）在每阶段明确出场与推进点；关键支线角色以标志性技能/视觉符号轮换出场，避免长时间断线。  
- 制作标注用于商业策略：首集/第3–4集为引流/试看节点；Ep30、Ep48、Ep59 为主付费/转化节点；少数集标注为“引流+付费双重剪片优先级”。

2. 分集 / 分段绑定清单（集数 1–60，按阶段分组；每集项包含：所属阶段 / 核心驱动人物 / 名场面等级 / 绑定名场面 / 单集纯剧情概要 / 节奏亮点 / 结尾钩子 / 制作标注（引流/付费/过渡）/ 人物推进说明）
（为便于现场制作与脚本组使用，按集逐条列出；每项短句化，保持可直接套用至分镜/台本说明）
