### 角色圣经与人物驱动说明

#### 一、核心人物设定与关系

**1. 萧狼**

- 身份 / 功能定位：主角。轨迹：普通高中生 → 校园老大 → 城市级地下势力领袖（狼帮首脑）→ 主动隐退者。全季一切冲突与选择围绕他展开。
- 核心性格标签：护短、冷静、重情、决断狠、有掌控欲、天然领袖感。
- 欲望 / 目标：
  - 前期：保护妹妹/身边人不受欺负，在校园小江湖保有话语权。
  - 中期：通过扩张地盘与资源，建立可控安全网，对抗外部威胁和“名单”。
  - 终局：在看清代价后，守住爱情与良知，从权力顶点脱身。
- 外部冲突：
  - 与校园恶霸、天海会、江南会的逐级对抗。
  - 与高层“牙/主宰”名单制度的博弈（随时可能被清洗）。
  - 面对警方、舆论与社会秩序的压力（尤其武大枪响后）。
- 内部矛盾 / 成长点：
  - 保护欲 vs 权力扩张：越强越危险，他不断意识到自己也是加害结构一部分。
  - 信任 vs 控制：对柳老大、吴勇、QA等的信任边界；对兄弟既依赖又控制。
  - 复仇冲动 vs 道德底线：武大枪响后是走极端屠杀还是止步于“只护短不滥杀”。
- 与主角关系：自我对抗型主角，内心戏多，所有重大决策集均需给他“选择镜头”。
- 与其他关键人物关系：
  - 柳老大：赏识+试探+交棒的复杂师徒/父辈关系。柳老大像“制度化的老狼”，萧狼是“新一代变种”。
  - 柳初夏：唯一能真正影响他价值观的情感锚。她一句话能左右萧狼的风险承受度。
  - 宿舍兄弟（远/慕容风/虞飞）：情绪家人和战斗中枢，是他“护短”原则的具体对象。
  - 吴勇：危险的情报源。萧狼既借他上升，也被他牵引到更黑的盘面。
  - 冷刀/火蛇：职业主义镜像——告诉他“纯职业不谈道德”的另一种活法。
  - 江南会/王浩：代表腐朽特权与无规则暴力，是他走上全面战争的导火线。
- 高光点 / 爆点：
  - 废厂成名战：从普通学生跨入“狼”的姿态。
  - 武大枪响后的失控与宣言：道德裂缝完全显形。
  - 杀蛇镇谈判：和职业雇佣兵平等博弈，显示智谋与胆量。
  - WH三河大战指挥：军事化部署和牺牲决断。
  - 大结局隐退：明知可一统江湖却选择放手。
- 短剧需要强化特质：
  - 眼神和短句：台词不宜太多，多靠眼神、动作给“狠”和“护短”。
  - 决策节奏：关键场面给到他2–3秒沉默+急切动作的反差。
  - 对弱者的温柔：与梦颜、柳初夏、牺牲兄弟家属的细腻互动。
- 阶段性变化 / arc：
  - A段：被动反击→自发组织。
  - B段：学会谈判和利用大哥资源。
  - C段：职业化、军事化，心理开始撕裂。
  - D段：彻底成“战场统帅”，逐渐看清规则残酷。
  - E段：承担代价，拒绝成为下一个“牙”，选择退出。

**2. 柳老大（柳流）**

- 身份 / 功能定位：流星会会首，本地秩序代表；萧狼的半导师+势力背书；也是权力道德线的“老一代样本”。
- 核心性格标签：稳、审时度势、重视家人、现实主义。
- 欲望 / 目标：维持本地平衡，不被更高层清洗，同时寻找到接班人，让妹妹安全离开江湖。
- 外部冲突：与天海会/江南会博弈；时刻提防“牙/主宰”的名单清洗。
- 内部矛盾 / 成长点：在“自己干到底”与“交棒给新狼”之间拉扯；既怕萧狼太狠又知道只有这种人能活下来。
- 与主角关系：亦父亦师亦潜在对手。前期观察，中期扶持，末段真正放手。
- 与其他关键人物关系：柳初夏的保护者；与吴勇有不明旧线（建议保留模糊）；对水子涵有“旧江湖看新势力”的评价视角。
- 高光点：
  - 别墅初会萧狼：从眼神中读人。
  - 白卡揭示：让观众第一次理解高层规则。
  - 交权与退位：权杖/戒指/烟等象征物交给萧狼。
- 短剧强化要点：
  - 台词少而精准，多用停顿和视线。
  - 出场场景固定在别墅、办公室等“权力空间”，形成记忆点。
- Arc：掌控者 → 挑选继承人 → 放手的老人。

**3. 柳初夏**

- 身份 / 功能定位：柳老大之妹；主情感线和道德锚，是萧狼“退出系统”的核心动力。
- 核心性格标签：温柔、清醒、有原则、有忍耐但不软弱。
- 欲望 / 目标：在乱局中保持普通人的生活可能性，不让萧狼彻底变成“只会杀人的工具”。
- 外部冲突：被用作筹码、被迫转移、成为攻击点。
- 内部矛盾 / 成长点：从旁观江湖到参与抉择，在“离开他=安全”与“留在他身边=共同背负”之间选择。
- 与主角关系：恋人+制动器。她的“安全感”是萧狼所有判断的一条重要约束。
- 高光点：
  - 首次在饭店遇险被护，反差好感。
  - 武大枪响后的视频/通话，对萧狼的质问。
  - 远走前的信和最后一同离开的画面。
- 短剧强化要点：
  - 给足近景和眼泪/笑的细节，情绪点为付费服务。
  - 她必须有主动行为（提出隐退、提出分手、提出条件），不能只是被保护物。
- Arc：旁观者 → 情感深度参与 → 推动主角隐退的核心力量。

**4. 刘天海（天海会头目）**

- 身份 / 功能定位：前中期主要对手之一，代表传统暴力派系与地盘逻辑。
- 核心性格标签：粗暴、好面子、地盘意识极强。
- 欲望 / 目标：扩大天海会影响力，压制流星会与新生狼帮。
- 外部冲突：与柳老大、萧狼直接对抗；被“名单制度”视为待清洗对象。
- 内部矛盾 / 成长点：并不理解高层政治，最终死于自己看不见的规则。
- 与主角关系：萧狼的中期开局对手，死亡间接促使萧狼进入更高层盘面。
- 高光点：
  - 与萧狼面对面对骂/对砍。
  - 被刺当晚的惊愕与不甘。
- 短剧强化要点：将其呈现为“旧时代狼王”，增强他的悲剧感，反衬萧狼新路。

**5. 吴勇**

- 身份 / 功能定位：情报中介/线人；极可能是“牙/主宰”的触须或外围。信息线和悬疑线的中枢。
- 核心性格标签：世故、滑头、模糊、不彻底可信。
- 欲望 / 目标：活下去并从信息垄断中获利，可能寻求在高层与本地之间腾挪一条生路。
- 外部冲突：各方猜疑、被跟踪和清算风险极高。
- 内部矛盾 / 成长点：在“继续当狗腿”与“提醒这些年轻人别死得太难看”之间摇摆。
- 与主角关系：模糊的帮手/威胁。每一次出现都要让萧狼做“信/不信”的选择。
- 高光点：
  - 白卡初次递交时的说话方式。
  - 被偷偷拍下与高层接头。
  - 失踪/牺牲前留下最后白卡。
- 短剧强化要点：
  - 台词总要带双关，观众能二刷体会另一层含义。
  - 每次出场都带一个新的信息点或反转，不能闲聊。

**6. 远**

- 外部冲突：面对枪械、雇佣兵、城市巷战的压力。
- 内部矛盾 / 成长点：从校园混战适应到现代战场，心理因战损逐步承压。
- 与主角关系：执行者+私人友谊，默默守在第一线。
- 高光点：
  - 夜袭中的声源判断镜头（耳朵特写）。
  - WH巷战中带队突围。

  - 慕容风：冷静、精准、少言。
  - 虞飞：嘴碎、有梗、冲动但很铁。
- 功能：活动气氛、承接局部牺牲、给团队提供人情味。
- 高光点：夜袭中一冷一热两种进攻逻辑；战前斗嘴战后一人牺牲的情绪爆发。

**8. 梦颜**

- 身份 / 功能定位：校园女神+重大受害者线；萧狼道德裂缝的触发器。
- 核心性格标签：温婉、脆弱、受伤后努力坚强。
- 欲望 / 目标：从创伤中走出，保留一点普通人的尊严。
- 外部冲突：被王浩骚扰、被枪击事件牵连。
- 内部矛盾 / 成长点：在“让一切过去”和“面对真相”之间徘徊。
- 高光点：武大枪响后的崩溃与后续告别。
- 短剧强化要点：对她的呈现要少暴力细节、多后果与心理，避免敏感风险。

**9. 王浩**

- 身份 / 功能定位：江南会少爷；特权与恶意的集中体现；推动江南会线升级。

**11. 冷刀 / 火蛇（QA精英）**

- 身份 / 功能定位：职业雇佣兵；冷刀近战，火蛇狙击。

- 欲望 / 目标：赚到足够的钱，保持团队名声。
- 功能：职业化战争范式的代表，与萧狼草莽对比。
- 高光点：演示战；WH决战中关键击杀。
- 短剧强化要点：造型和动作统一风格，构成可单独营销的“雇佣兵IP”。

**12. 水子涵**

- 身份 / 功能定位：一河老大，后期重要盟友与潜在新管理层。
- 核心性格标签：精明、算计、趋利避害但有底线。
- 欲望 / 目标：保地盘、保自己，在新秩序中占一席。
- 功能：提供“谈判江湖”的视角，展示萧狼权力整合能力。
- 高光点：被救援后转向合作；最后握手合并一河。
