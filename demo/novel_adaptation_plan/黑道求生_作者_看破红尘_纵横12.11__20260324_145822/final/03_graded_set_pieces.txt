1）分级规则确认
- 参照既定规则对每个候选名场面逐一评级：
  - S 级：满足 ≥3 项高价值特征，且属于主线终极节点 / 全局反转 / 极致情绪（通常 1 个 = 1 集）。
  - A 级：满足 ≥2 项特征，且属于主线关键节点 / 中强高光（通常 1–2 个 = 1 集）。
  - B 级：满足 ≥1 项特征，属于爽点 / 情绪爆发 / 主线高潮片段（通常 2–3 个 = 1 集）。
  - C 级：无上述高价值特征，仅作剧情衔接或过渡，不占名场面名额。
- 评判维度取自底稿的“后续提取名场面的重点观察维度”：叙事功能、冲突强度、视觉/听觉记忆点、角色动机显化、情绪落点/代价感、可拆分性、敏感度与延展悬念性，并结合短剧传播和竖屏切片适配度。

2）S级名场面清单（必须作为单集或付费/高潮位保障）
- 武大宿舍枪响·梦颜倒下（编号4）
  - 理由：极致情绪与道德裂缝（情绪极值）；主线不可逆节点（触发萧狼从“护短”到全面升级的选择）；强传播钩（受害→宣言→视频扩散）；三段式结构完备，短剧化后可作为重要付费/转折集。
- WH 三河大战 / 江南会终极对决（来自剧情骨架）
  - 理由：全剧动作高潮、多人关係与势力对撞的终极爆发（主线终极节点）；视觉记忆强（巷战、爆破、狙击）；带来大量代价感（牺牲/势力崩盘），天然成为季终/重磅付费点。
- 狼帮成立 → 代价清算 → 萧狼选择隐退（结局三连段，来自剧情骨架）
  - 理由：情感闭环与价值观终极抉择（全局反转/情绪收束）；具备强烈代价感（兄弟牺牲、创伤、交权）；是结局钩与余味悬念的复合场面，适合作为最终一集或多集浓缩的付费/宣传高潮。

3）A级名场面清单（关键节点，优先上中段承接与付费卡位）
- 废弃工厂成名战——少年变“狼”（编号2）
  - 理由：势力雏形与人设符号化（“狼”围巾）；主线关键节点（势力形成）；视觉与情绪钩强，便于短片化传播。
- 别墅交棒 / 阎王帖·白卡首次揭示（编号3）
  - 理由：权力交接的仪式感 + 高层威胁信息曝光（剧情钩、悬念）；推动剧情从街头到权谋的尺度跃迁，适合作为中期付费/重点集。
- 饭店护妹——第一道底线（编号1）
  - 理由：人物人设钩（护短底线）与公开对峙；形成首集或前几集的强传播钩，能快速抓住受众并引入高层注视线索。
- 杀蛇镇军火议价·QA初现（编号5）
  - 理由：外线职业化转折（从草莽走向合同化战争）；异域视觉与职业冷感，推动叙事进入C阶段的关键转折。
- 刘天海被刺（剧情骨架项）
  - 理由：推动地域势力重组的关键触点，是 B→D 段迈进的重要节点，具有强剧情驱动与后续连锁效果。

4）B级名场面清单（爽点/动作/情绪爆发，但非必须占单独名场面名额）
- 收编场子（KTV/迪厅）类战斗/谈判片段（阶段B 中多处可拆分的爽点）
  - 理由：具动作爽感与谈判戏，易剪成宣传短片，但对全局不是独立终极点。
- 远 / 慕容风 / 虞飞 等突击组的夜袭 / 斩首行动（局部高潮）
  - 理由：动作亮点、团队高光；适合作为连续剧集中多处高潮或夹点。
- QA 营地展示 / 冷刀火蛇的单次斩首或狙击节段
  - 理由：职业化视觉与动作美学，但属战术级别的爽点。
- 水子涵被救援后的谈判/合并小高潮
  - 理由：谈判戏提供智谋爽点与权力整合感，但通常与更大的战役相连。

5）C级过渡片段清单（仅作剧情衔接，不计入名场面名额）
- 校园统一后常态化的宿舍日常（团队磨合、训练短段）
- 情报传递/手机视频扩散的程序化片段（校园舆论/警务流转的中转镜头）
- 当地行政/警方插入的流程化处置镜头（救护车、警戒线、证物收集等）
- 小规模谈判或物资转运的过桥段（未产生反转或强视觉记忆的交易细节）
- 侧支人物的短暂内心剖白/配角回忆镜头（如非造成主线走向改变的个体情绪段落）

6）分级统计
- S 级数量：3（武大宿舍枪响·梦颜倒下；WH三河大战/江南会终极对决；狼帮成立→代价清算→萧狼隐退）
- A 级数量：5（废弃工厂成名战；别墅交棒/阎王帖揭示；饭店护妹；杀蛇镇军火议价；刘天海被刺）
- B 级数量：4（收编场子/KTV 战斗、突击组夜袭/斩首、QA 营地展示、水子涵谈判小高潮）
- C 级数量：5+（宿舍日常、情报/手机视频扩散程序化片段、警方流程、物资转运过桥、配角短剖白 等）

（注：B/C 为示范型类目，实际制作中可把若干 B 片段合并成单集高密度爽点；C 类主要作为节奏缓冲，不单独占名场面额度。）

7）落地建议（优先级与布置位）
- 开篇/首集（强吸引钩）
  - 必占：饭店护妹（编号1）——作为首集或 1–2 集内的开场钩子，建立“护短”人设与高层注视，适合制作 30–90 秒竖屏宣传短片。
  - 次优：废弃工厂成名战（编号2）——放在序章末或第2–4 集，快速确立势力雏形与“狼”符号，制造社群讨论与角色认同。
- 中段付费/卡位（保留观众并制造转折）
  - 必占：别墅交棒 + 阎王帖/白卡揭示（编号3）——中期重要付费点/付费卡位（悬念暴露），建议做为跨集内容（1 集主线 + 1 集延伸影像/回放）。
  - 优先考虑：杀蛇镇军火议价（编号5）——作为“职业化升级”展示，可设置为付费或专题推广素材（异域视觉易传播）。
- 高潮/付费顶位
  - 必占：武大宿舍枪响（编号4）——作为重磅付费集或付费预告核心（强情绪爆发），务必在剪辑与分镜上处理好敏感度（对暴力/受害呈现以镜外或象征替代）。
  - 必占：WH 三河大战 / 江南会终极对决（季终）——作为整季动作高潮与付费押宝，投入场面调度、爆破特效与音乐铺陈。
- 结尾/闭环（情感收束与余味）
  - 必占：狼帮成立→代价清算→萧狼隐退（大结局）——留作季终最终集或两集连播，作为情感闭环与续季钩（“牙/主宰”线未决）共同出现。
- 宣传与短视频切片建议
  - 动作爽点（废厂成名、夜袭、WH 决战）做 15–45 秒竖屏循环片段，用“动作→停格→口白（金句）”格式吸引流量；情感钩（饭店护妹、武大宿舍枪响后萧狼的低语）做 15–30 秒情感短片，投放在社交付费与首刷前。
- 敏感度/合规提示（短评）
  - 武大宿舍枪响与梦颜受创等涉及暴力/创伤的场面须弱化直接暴露（采用镜外、特写物件、声音剪接与人物反应替代直呈血腥）；任何可能触及性暴力或未成年受害的镜头应优先用暗示与后果呈现而非细节描摹。
- 采编落地建议（便于后续节奏规划）
  - 把 S 级场面作为“单集核心”分配（每个 S 场面保留 1 集完整呈现）；A 级场面作中段关键集（可与若干 B 段合并）；B 级作为集内多点爆发与短视频素材池；C 级用于过渡与节奏缓冲，避免占用名场面名额。
  - 在做 60 集拆分时，优先把 A/S 场面均匀分布在第 1–45 集以维持张力，第 46–60 集集中放 WH 决战、代价清算与隐退闭环（并在第 30–40 集设置别墅交棒/阎王帖的重大反转钩）。

结语
- 本次分级已把底稿中所有关键候选与剧情骨干的高价值节点纳入评估，S/A/B/C 四档分配兼顾叙事重量与短剧传播适配性。下一步可基于此分级结果进行“5 阶段节奏规划 + 单集名场面绑定”，将每个 S/A/B 场面具体绑定到 60 集架构中的集数与每集钩子（如需我可继续生成草稿）。
