## 概览
本指南提供AI解说剧集数推荐的标准和依据，确保每集1-2分钟的节奏合理。

## 集数计算标准

### 基础参数
- **单集时长**:1-2分钟（标准推荐1.5分钟）
- **单集字数**:约300-400字（正常语速下）
- **信息密度**:每集覆盖1-2个关键情节点

## 影响因素

### 1. 文档字数
| 字数范围 | 基础集数 |
|----------|----------|
| 2000-5000字 | 5-8集 |
| 5000-10000字 | 8-15集 |
| 10000-20000字 | 15-25集 |
| 20000-50000字 | 25-40集 |
| 50000字以上 | 40集以上 |

### 2. 情节密度
- **高密度**（情节紧凑，转折多）:集数增加20-30%
- **中密度**（情节适中，节奏平稳）:按标准计算
- **低密度**（情节舒缓，描述多）:集数减少15-25%

### 3. 人物复杂度
- **复杂**（主要角色>5个，关系复杂）:集数增加15-25%
- **中等**（主要角色3-5个）:按标准计算
- **简单**（主要角色<3个）:集数减少10-15%

### 4. 故事类型
| 故事类型 | 调整系数 |
|----------|----------|
| 悬疑推理 | +20-30%（需要铺垫和反转） |
| 爱情故事 | ±0（按标准计算） |
| 成长励志 | +10-15%（情感需要渐进） |
| 奇幻冒险 | +15-25%（世界观需要解释） |
| 日常轻喜剧 | -10-15%（节奏较快） |

## 推荐公式

```
基础集数 = 文档字数 / 400

情节密度系数:
  高密度 = 1.25
  中密度 = 1.0
  低密度 = 0.8

人物复杂度系数:
  复杂 = 1.2
  中等 = 1.0
  简单 = 0.9

故事类型系数:
  根据上表选择对应系数

最终推荐集数 = 基础集数 × 情节密度系数 × 人物复杂度系数 × 故事类型系数
```

### 计算示例
某科幻小说:
- 字数:15000字
- 情节密度:高（1.25）
- 人物复杂度:中等（1.0）
- 故事类型:奇幻冒险（1.2）

计算:
```
基础集数 = 15000 / 400 = 37.5集
推荐集数 = 37.5 × 1.25 × 1.0 × 1.2 = 56.25集 → 推荐约55-60集
```

## 调整原则

### 增加集数的场景
1. 情节转折多，需要详细解释
2. 人物关系复杂，需要铺垫
3. 世界观设定丰富，需要介绍
4. 情感戏份重，需要渲染

### 减少集数的场景
1. 情节直白，无需过多解释
2. 人物少且关系简单
3. 节奏快，信息密度高
4. 注重悬念而非铺垫

### 特殊情况
1. **系列化作品**:建议拆分为多个独立系列，每个系列20-30集
2. **开放式结局**:可增加1-2集作为收尾
3. **番外内容**:可作为额外3-5集补充

## 注意事项
- 推荐集数为估算值，实际制作中可根据观众反馈调整
- 每集控制在1-2分钟，确保观众完播率
- 前3集是关键，建议在前3集中制造高潮或悬念
- 避免集数过多导致观众疲劳，超过100集建议拆分系列
