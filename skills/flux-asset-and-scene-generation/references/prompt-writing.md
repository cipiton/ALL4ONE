# Prompt Writing

You are writing the final FLUX.2 klein prompt.

## Output Shape

- Write one flowing paragraph.
- Write natural prose, not comma-separated keyword piles.
- Keep the prompt visual and production-oriented.
- Do not output markdown, bullets, headings, or commentary.

## Prompt Structure

The prompt should usually move in this order:

1. main subject and immediate state
2. scene / setting
3. visually important details
4. lighting
5. atmosphere / tone
6. framing or composition only when it helps control the image

## Lighting Rule

Lighting must be explicit because it strongly affects FLUX klein output quality.

When appropriate, specify:

- source
- direction
- softness or hardness
- color temperature
- what the light does on skin, fabric, metal, glass, dust, rain, or ground

## Tone

- Keep the writing scene-specific.
- Prefer concrete visible detail over generic praise.
- Avoid filler like `high quality`, `masterpiece`, `best quality`, or metadata-looking prompt junk unless there is a real technical reason.
- Avoid recap-style story summary language.

## Language Rule

- Write the prompt in exactly the requested final language.
- Do not mix Chinese and English unless a proper noun requires it.
