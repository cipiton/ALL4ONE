# Step 2 Prompt: Build Sanitization Plan

你现在执行的是「剧本洗稿」流程的第 2 步：生成替换与规避方案。

你会收到：

- 规范化原稿
- 敏感词参考资料

你的任务是基于规范化原稿，为后续洗稿执行输出一份清晰、可直接应用的方案。

## 目标

建立一份实用的洗稿执行方案，覆盖：

1. 人物名替换
2. 地名/组织名/公司名替换
3. 关键道具名替换
4. 系统/契约/能力/头衔/身份标签替换
5. 敏感词规避与替换策略
6. 风格统一原则

## 输出要求

请输出纯文本方案，结构必须清晰，建议使用以下结构：

### 洗稿目标
### 原稿类型与风险概况
### 角色名替换建议
### 地名与组织名替换建议
### 关键道具名替换建议
### 系统/能力/头衔替换建议
### 敏感词替换建议
### 风格与语气统一规则
### 执行注意事项

## 方案标准

- 建议必须具体，不要空泛
- 替换建议必须符合原作世界观与题材
- 中文项目默认给出中文到中文的刷新方案，不要把核心名词转成英文或拼音主名
- 敏感词处理要结合上下文，不要机械替换
- 如果某些词不宜统一替换，必须注明条件
- 方案要足够清楚，能直接指导第 3 步洗稿执行

## 禁止事项

- 不要输出最终洗稿剧本
- 不要只列敏感词而缺少执行策略
- 不要丢失关键人物或关键道具
- 不要输出 JSON

输出应当是一份可被确认和执行的完整洗稿方案。
