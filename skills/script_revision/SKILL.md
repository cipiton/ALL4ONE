# 剧本洗稿

## 任务目标

- 本 Skill 用于：剧本洗稿与内容改编
- 能力包含：智能名称替换、敏感词处理、保留剧情逻辑的洗稿输出
- 触发条件：用户提供剧本需要洗稿、重写或改编

## 操作步骤

### 步骤一：获取并阅读完整原文

**强制要求：必须完整阅读完小说/剧本原文，才能开始后续操作。**

1. 询问用户剧名
   - 若用户未输入剧名，提供 3-5 个创意剧名建议供选择
   - 建议风格需符合剧本类型（古风/现代/悬疑等）

2. 获取并完整阅读原文内容
   - 请用户提供完整小说/剧本文本
   - 支持分段提供或整体上传
   - **必须确认已完整阅读所有章节/段落**，不可中途开始处理

### 步骤二：生成洗稿方案（文本形式）

**结合原文内容和用户提供的改编方案，综合生成洗稿方案。**

**方案内容：**

1. 名称识别与替换建议
   - 角色名：所有出场人物，包括主角、配角、路人
   - 地名：城市、街道、建筑、区域等
   - 公司名：企业、组织、机构名称
   - 关键道具名：与剧情推进或主角人设密切相关的物品

2. 敏感词识别与替换
   - 基于 [references/sensitive-words.md](references/sensitive-words.md) 扫描识别
   - 提供替换建议

3. 改编方向调整（如有用户提供的改编方案）
   - 根据改编需求调整名称风格
   - 根据改编需求调整敏感词处理策略

**方案输出格式（纯文本）：**
```
========== 洗稿方案 ==========

【原文概况】
剧名：《XXX》
字数：约 X 万字
类型：古风/现代/悬疑等

【角色名替换建议】
原名：张三 → 建议替换：李墨尘 / 顾青云 / 沈无忌
原名：李四 → 建议替换：王逸风 / 陆长歌 / 萧寒山

【地名替换建议】
原名：青云市 → 建议替换：沧澜市 / 凌霄市 / 碧落市

【公司名替换建议】
原名：盛世集团 → 建议替换：鼎盛集团 / 锦绣集团 / 华鼎集团

【关键道具名替换建议】
原名：龙纹玉佩 → 建议替换：凤鸣玉佩 / 麒麟玉佩 / 苍龙玉佩

【敏感词替换】
"鬼" → "诡"（共 X 处）
"妖魔" → "邪异"（共 X 处）
"杀人" → "害人"（共 X 处）

【改编说明】（如有）
根据改编方案，本次洗稿将...

==============================

请确认以上洗稿方案，或提出修改意见。
确认后发送"洗稿剧本"指令开始执行洗稿任务。
```

**输出规则：**
- 方案仅以纯文本形式输出，不生成文档文件
- 用户未确认前，不进行洗稿执行
- 用户未发送"洗稿剧本"指令前，仅停留在方案阶段

### 步骤三：用户确认方案

**用户可操作：**
- 确认方案：回复"确认"或"洗稿剧本"
- 修改方案：指出需要调整的替换建议，重新生成方案
- 自定义替换：指定特定的替换名称
- 要求重新生成建议

**确认后行为：**
- 用户确认方案后，方案内容可保存为文档
- 等待用户发送"洗稿剧本"指令

### 步骤四：执行洗稿任务

**触发条件：用户发送"洗稿剧本"指令**

用户发送指令后，执行洗稿输出：

1. 应用所有确认的替换
2. 保持原文格式（对话、场景描述、旁白等）
3. 检查上下文一致性
4. 输出完整洗稿内容

**输出要求：**
- 保留原剧本的结构和节奏
- 确保替换后语句通顺
- 验证关键道具名在全文中统一替换
- 检查敏感词是否全部处理

**输出形式：**
- 直接输出洗稿后的文本内容
- 可根据用户需求保存为文档

## 工作流程图

```
用户提供原文 → 完整阅读原文 → 生成洗稿方案（文本）
                                    ↓
                              用户确认方案
                                    ↓
                           用户发送"洗稿剧本"指令
                                    ↓
                              执行洗稿输出
```

## 注意事项

- **必须完整阅读原文**：不可在未读完原文的情况下生成方案
- **方案结合改编需求**：如有用户提供改编方案，需综合考虑
- **指令触发执行**：仅当用户发送"洗稿剧本"指令后才执行洗稿任务
- **方案文本形式**：用户未确认前，方案仅以文本输出，不生成文档
- 仅替换关键道具名，普通道具无需替换
- 敏感词替换需考虑语境，避免生硬替换影响阅读体验
- 姓名替换时注意避免与历史名人重名
- 地名替换需保持世界观一致性
