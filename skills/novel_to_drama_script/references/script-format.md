## 1. 基本格式要求
- 字数：每集1000-2000字
- 对白占比：>70%
- 单句对白：≤12字（吟唱/念咒等特殊场景除外）
- 场景划分：根据情节需要划分场景，每个场景有明确的时间、地点
- 人物介绍：新人物首次出场时要有简要介绍

## 2. 短剧创作规律

### 2.1 第一集黄金开头
- 前3秒必须吸睛，抓住观众注意力
- 可以使用：悬念、冲突、震撼画面、精彩台词等
- 让观众立刻产生"想知道接下来发生什么"的欲望

### 2.2 每集开头承接
- 呼应或承接上一集结尾的钩子/悬念
- 让观众有连贯感，不会脱节
- 可以用：回忆镜头、人物对话回顾、直接延续场景等方式

### 2.3 每集结尾钩子
- 结尾必须留钩子或悬念
- 让观众想看下一集
- 钩子类型：
  - 新危机出现
  - 秘密即将揭开
  - 人物关系剧变
  - 意外转折
  - 关键问题 unanswered

### 2.4 情绪拐点
- 每集中间必须有情绪拐点或冲突节点
- 类型包括：
  - 反转：剧情走向意外改变
  - 高潮：情绪或冲突达到最高点
  - 打脸：反派被打脸，正义得到伸张
  - 揭秘：重要秘密被揭露
  - 决裂/和好：人物关系重大变化

## 3. 场景描述规范
格式：
```
【第X集】
【场景X】
时间：XXXXX
地点：XXXXX
```

要求：
- 时间要明确（如：清晨、黄昏、深夜等）
- 地点要具体（如：茶馆二楼、书房内、郊外树林等）
- 每个场景要有明确的主题和作用

## 4. 人物对白规范
格式：
```
人物名：（动作/表情描述）对话内容
```

要求：
- 人物名加粗或用冒号隔开
- 动作和表情描述放在括号内，要简洁有力
- 对话要符合人物性格和身份
- 对话要自然流畅，避免生硬
- **单句对白≤12字**（吟唱/念咒等特殊场景除外）
- 大段对白必须拆分，中间穿插画面描述

### 对白拆分示例
❌ 错误（大段对白）：
```
**张三**：我知道你今天来这里的目的，你不就是想知道那个秘密吗？其实那个秘密我已经守了很多年了，今天我就告诉你吧。
```

✅ 正确（拆分+画面穿插）：
```
**张三**：（盯着对方）我知道你的目的。
△ 李四瞳孔微微收缩。

**张三**：你想知道那个秘密。
△ 窗外的风突然变大。

**张三**：我守了很多年。
△ 茶杯里的水泛起涟漪。

**张三**：今天，我告诉你。
```

## 5. 画面描述规范
格式：
```
△ 画面描述内容
```

### 画面类型
1. **环境空镜**：描述环境，营造氛围
   - 示例：△ 夕阳染红了整个天空。
   
2. **反应镜**：描述人物的反应
   - 示例：△ 她的眼眶瞬间红了。
   
3. **特写**：特写某个细节
   - 示例：△ 手指紧紧攥着衣角。
   
4. **动作镜头**：描述动作过程
   - 示例：△ 他猛地拍案而起。

要求：
- 语言要简洁生动，有画面感
- 不要过长，重点突出关键信息
- 大段对白之间必须穿插画面
- 画面描述要服务于情绪表达和剧情推进

## 6. 完整示例

【第1集】
【场景1】
时间：深夜
地点：破庙

△ 【黄金开头】一道闪电划破夜空，照亮破庙内一具倒在血泊中的尸体。

**张三**：（冲进来）发生什么了？
△ 他的脚步声在空荡的破庙里回响。

**李四**：（指着尸体）他...他死了。
△ 李四的声音在颤抖。

**张三**：（蹲下查看）什么时候？
△ 尸体的胸口插着一把刀。

**李四**：就刚才...我...
△ 窗外传来诡异的笑声。

**张三**：（猛地抬头）谁？！
△ 【钩子】破庙的门"吱呀"一声自己开了。

（字数约300字，对白占比约40%，单句对白均≤12字）
