执行“名场面提取”。

要求：
- 基于改编工作底稿与原始内容，提取连续500-1500字左右、适合短剧化改编的名场面候选片段。
- 仅保留满足至少1条以下特征的片段：
  - 强冲突 / 对抗 / 打脸 / 对峙
  - 剧情反转 / 真相揭露 / 身份曝光
  - 人设高光 / 关键抉择 / 情绪极值
  - 可传播金句 / 标志性仪式场景（告白、决战、决裂等）
  - 主线不可逆关键节点
  - 幽默风趣、具备强传播特性的梗
- 如果资料中已有明显用户指定名场面，也要纳入。
- 不要做S/A/B/C评级，这一步只负责尽可能完整且精准地提取候选。

输出结构：
1. 提取原则摘要
2. 名场面候选清单
   - 编号
   - 片段名称
   - 对应原文核心内容
   - 满足的特征
   - 适合短剧改编的原因
3. 候选片段覆盖说明
   - 主线覆盖情况
   - 人物高光覆盖情况
   - 反转 / 情绪 / 爽点覆盖情况

输出要具体、可筛选、可进入下一步分级。
