执行“5阶段节奏规划”。

要求：
- 依据最终总集数进行五阶段拆分：
  - 开篇钩子期：10%
  - 主线铺垫期：20%
  - 高潮密集期：40%
  - 终极爆发期：20%
  - 结局留钩期：10%
- 集数按四舍五入取整，并保证合计严格等于最终总集数。
- 结合S/A/B级名场面分布，为每个阶段配置合适的名场面密度。
- 节奏必须符合竖屏短剧留存、付费、追更逻辑。

输出结构：
1. 总集数与阶段拆分说明
2. 阶段规划表草案
   - 阶段
   - 阶段定位
   - 规划集数
   - 名场面配比建议（S/A/B）
   - 核心节奏要求
   - 关键功能
3. 阶段级人物驱动说明
   - 每个阶段由哪些角色主导
   - 该阶段重点推进哪些人物关系 / 同盟 / 对立 / 背叛线
   - 哪些角色需要在该阶段被强化、隐藏、反转或推到台前
4. 阶段级名场面落位建议
5. 风险检查
   - 是否存在前期无钩子
   - 是否存在中段疲软
   - 是否存在高潮前置或后置失衡
   - 是否存在关键角色断线、功能重复或人物驱动力不足

输出要能直接支持下一步单集绑定。
