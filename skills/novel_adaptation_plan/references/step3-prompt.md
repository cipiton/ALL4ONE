执行“名场面分级”。

要求：
- 依据以下规则对候选名场面逐一评级：
  - S级：满足≥3条特征，且属于主线终极节点 / 全局反转 / 极致情绪，通常1个=1集
  - A级：满足≥2条特征，且属于主线关键节点 / 中强高光，通常1-2个=1集
  - B级：满足≥1条特征，属于爽点 / 情绪 / 主线高潮，通常2-3个=1集
  - C级：无上述高价值特征，纯过渡，仅做衔接，不占名场面名额
- 分级时要结合题材特点、主线权重、商业传播性、短剧适配度。
- 不要跳到分集规划，只输出分级结果与简要理由。

输出结构：
1. 分级规则确认
2. S级名场面清单
3. A级名场面清单
4. B级名场面清单
5. C级过渡片段清单
6. 分级统计
   - S级数量
   - A级数量
   - B级数量
   - C级数量
7. 落地建议
   - 哪些名场面必须优先占据开篇 / 付费点 / 高潮位

输出必须可直接给后续节奏规划使用。
