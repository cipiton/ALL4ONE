执行“单集名场面绑定”。

要求：
- 结合名场面分级结果与5阶段节奏规划，把高价值内容绑定到具体集数或具体集段。
- 每集必须具备以下三者之一：冲突 / 反转 / 情绪峰值。
- 每集结尾必须设置强钩子：悬念 / 爽点预告 / 伏笔。
- S级名场面必须优先落在开篇、付费点、终极高潮位。
- 过渡剧情总占比不得超过10%。
- 若资料无法支撑逐集极细拆分，可按集段总包，但最终必须足够支持标准化表格输出。

输出结构：
1. 单集绑定原则
2. 分集 / 分段绑定清单
   - 集数或集段
   - 所属阶段
   - 核心驱动人物 / 关键人物组合
   - 名场面等级
   - 绑定名场面
   - 单集纯剧情概要
   - 节奏亮点
   - 结尾钩子
   - 制作标注（引流 / 付费 / 过渡）
   - 人物推进说明（出场 / 升级 / 反转 / 隐藏 / 关系推进）
3. 覆盖校验
   - S/A级名场面是否已全部落地
   - 过渡占比是否达标
   - 钩子密度是否达标
   - 关键人物是否都有明确功能落点
   - 是否存在重要角色长时间消失或关系线断裂

输出必须尽量接近最终表2所需内容。
