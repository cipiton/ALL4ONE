# 步骤二：分集剧情

## 任务目标
参考已定稿的故事，符合短剧剧本的节奏要求，并按照用户集数输出每集分集剧情

## 核心要求

### 结构要求
1. **第一集特殊要求**：
   - 开头将整个故事最激烈的冲突情节前置
   - 制造强烈悬念
   - 然后按线性顺序展开剧情

2. **每集通用要求**：
   - 每集剧情有冲突/转折
   - 结尾有勾子（悬念）
   - 确保每集有看点
   - 衔接流畅
   - 适配短剧传播特点（节奏快、信息密集）

### 节奏要求
- 每集时长概念：按短剧节奏，通常每集1-3分钟
- 信息量：每集包含1-2个关键事件
- 冲突密度：每集都要有小冲突或转折
- 勾子强度：结尾勾子要足够吸引用户想看下一集

## 输出格式

### 严格参考附件
详细格式请参考 [assets/fractional-plot-reference.docx](assets/fractional-plot-reference.docx)

### 基本格式框架

```
## 第X集：《XXXX》（每集标题）

**核心事件**：一句话概括本集主要内容

**剧情详情**：
1. [具体情节1]
2. [具体情节2]
3. [具体情节3]

**本集看点**：
- [看点1]
- [看点2]

**结尾勾子**：[悬念内容，让用户想看下一集]
```

## 分集规划要点

### 整体规划
- 根据总集数合理分配剧情
- 开头几集要快速建立人物关系和核心冲突
- 中间集数要持续升级对抗，增加爽点
- 后几集要推向高潮
- 最后一集要给出 satisfying 的结局

### 分集衔接
- 上一集的勾子要在这一集开头解决
- 这一集的结尾要为下一集埋下伏笔
- 人物情绪和状态要连贯
- 剧情逻辑要连贯

### 爽点安排
- 每3-5集要有一个大爽点
- 每集要有小爽点或小转折
- 爽点要符合人物设定和剧情逻辑
- 爽点要有铺垫，不能太突兀

## 第一集特殊结构

### 倒叙开篇结构
```
## 第1集：《XXXX》

**开头（倒叙高潮）**：
[最激烈的冲突场景，如：女主当着所有人的面揭露反派真面目，反派气急败坏要报复，女主冷静反击]

**时间回到X天前**：
[正常顺序开始讲故事，建立人物关系，展示主角处境]

**本集结尾**：
[制造第一个悬念，如：女主发现了一个重要秘密，或反派即将实施一个阴谋]
```

## 注意事项
1. 严格按照用户要求的集数进行分集
2. 不要删减主线剧情，但可以适当调整节奏
3. 确保每集都有独立的完整性，同时又能勾连上下
4. 符合短剧传播特点：节奏快、冲突密集、信息量大
5. 保持人物性格的一致性
