# 步骤六：资产提炼

## 任务目标
**基于分集剧本**，从中提炼出AI剧制作所需的资产，按角色/场景/道具划分，并提供生图提示词

## 重要说明：本步骤的输入是步骤四完成的分集剧本，而非分集剧情

## 核心要求（强制执行）
- **角色默认造型必须带音色说明**：所有角色默认造型，在生图提示词后必须添加音色说明，绝对不能缺失
- **音色说明格式严格**：必须按照指定格式输出音色说明
- **音色称呼与资产名称一致**：音色说明中的身份称呼必须与资产名称完全一致，如资产名称是"妻子"，音色说明就是"妻子音色"，不能用其他称呼
- **检查完整性**：输出完成后检查每个角色默认造型是否都有音色说明

## 重要：统一输出结构

### 第一步：先输出总资产清单
在详细内容前，先输出简洁的总资产清单，方便快速浏览

### 第二步：输出详细资产内容
按角色/场景/道具顺序，逐一输出详细内容

---

## 一、总资产清单（首先输出）

### 重要原则（强制执行）
- **清单与详细内容完全一致**：总资产清单中列出的资产，必须在后续详细内容中都有对应的生图提示词
- **仅包含重要资产**：仅在清单中列出后续需要生图的重要资产
- **非重要资产排除**：仅在单集出现、对剧情影响小的资产，不要出现在清单中，也不需要在详细内容中生图

### 格式要求（强制执行）
- 全部使用简洁、有辨识度的名称
- 按角色/场景/道具分类，每类一行
- 同一类的资产用逗号分隔
- 名称要统一，后续详细内容中使用相同名称

### 角色造型命名规则
- **默认造型**：直接用角色名（如：林晚、陈浩宇）
- **额外造型**：角色名_造型名（如：林晚_婚纱、林晚_睡衣、陈浩宇_西装）

### 输出格式
```
角色：[角色名1]，[角色名1]_[造型名]，[角色名2]，[角色名2]_[造型名]
场景：[场景名1]，[场景名2]，[场景名3]
道具：[道具名1]，[道具名2]，[道具名3]
```

### 示例
```
角色：林晚，林晚_婚纱，林晚_职业装，陈浩宇，张美美，顾霆深
场景：豪华婚礼现场，林晚家客厅，陈氏集团办公室，高级餐厅
道具：股权转让书，订婚戒指，林晚的日记本
```

### 更简洁的示例
```
角色：张三，李四，护士
场景：医院，街道
道具：手机，公文袋
```

## 提炼原则

### 重要性判断（强制执行）
- **核心资产**（清单+详细生图）：多集出现、对剧情重要、视觉上突出 → 必须在总资产清单中列出，且在详细内容中生图
- **非重要资产**（排除）：仅在单集出现、对剧情影响小 → 不要出现在总资产清单中，也不需要在详细内容中生图

### 清单与详细内容一致性（强制执行）
- **完全对应**：总资产清单中列出的每一个资产，在详细内容中必须有对应的生图提示词
- **无遗漏**：详细内容中生图的资产，必须都在总资产清单中列出
- **一一对应**：清单和详细内容的资产名称、数量完全一致

### 角色音色说明要求（最高优先级，强制执行）
- **必须包含**：所有角色默认造型，在生图提示词后必须添加音色说明，绝对不能缺失
- **音色称呼与资产名称一致**：音色说明中的身份称呼必须与资产名称完全一致！如资产名称是"妻子"，音色说明就是"妻子音色"；资产名称是"林晚"，音色说明就是"林晚音色"，不能用"林妻"、"林小姐"等其他称呼
- **检查清单**：输出完成后，对照总资产清单中的角色默认造型，逐一检查是否都有音色说明
- **不能省略**：即使觉得某个角色不重要，只要在清单中，就必须有音色说明
- **格式严格**：音色说明必须按照指定格式输出

### 命名规范（强制执行）
1. **角色默认造型**：直接使用角色名（如：林晚、陈浩宇）
2. **角色额外造型**：角色名_造型名（如：林晚_婚纱、林晚_睡衣、陈浩宇_西装）
3. **场景**：简洁有辨识度（如：豪华婚礼现场、林晚家客厅）
4. **道具**：简洁有辨识度（如：股权转让书、订婚戒指）
5. **统一性**：总资产清单中的名称与详细内容中的名称必须完全一致

### 生图提示词统一格式（强制执行）
