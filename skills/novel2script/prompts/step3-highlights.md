# 步骤三：亮点梳理

## 任务目标
**基于分集剧情**（而非分集剧本），分析梳理整部剧的爽点、反转、打脸

## 重要说明：本步骤的输入是步骤二完成的分集剧情

## 梳理维度

### 一、爽点梳理

#### 爽点定义
爽点是让观众感到过瘾、解气、满足的情节

#### 爽点分类
1. **复仇爽**：主角对反派进行报复
2. **打脸爽**：反派或配角看不起主角，结果被狠狠打脸
3. **能力展示爽**：主角展现过人能力，让周围人震惊
4. **资源获取爽**：主角获得重要资源（金钱、权力、人脉等）
5. **情感爽**：主角和喜欢的人感情升温，或得到重要人物的认可
6. **真相揭露爽**：揭露反派的阴谋或谎言

#### 梳理要求
- 按时间顺序列出所有爽点
- 注明每一个爽点出现在第几集
- 分析爽点的类型
- 说明爽点如何满足观众心理

### 二、反转梳理

#### 反转定义
反转是剧情出现出乎意料但又在情理之中的转折

#### 反转分类
1. **身份反转**：人物身份出现意想不到的变化
2. **关系反转**：人物关系出现颠覆性变化
3. **局势反转**：原本劣势的一方突然占据优势
4. **信息反转**：新信息出现，彻底改变对之前事件的理解
5. **动机反转**：人物行为动机出现意外解释

#### 梳理要求
- 按时间顺序列出所有反转
- 注明每一个反转出现在第几集
- 分析反转的类型
- 说明反转的铺垫和合理性
- 分析反转带来的戏剧效果

### 三、打脸梳理

#### 打脸定义
打脸是指某人狂妄自大或看不起人，结果被事实证明错误，颜面扫地

#### 打脸分类
1. **实力打脸**：看不起主角实力，结果被主角实力碾压
2. **智商打脸**：觉得主角愚蠢，结果被主角智商玩弄
3. **判断打脸**：对事情判断错误，结果事实相反
4. **地位打脸**：用身份地位压人，结果发现对方身份更牛

#### 梳理要求
- 按时间顺序列出所有打脸情节
- 注明每一个打脸情节出现在第几集
- 分析打脸的类型
- 说明被打者之前的狂妄表现
- 说明打脸的具体方式和力度
- 分析打脸带来的爽感

## 输出格式

### 整体结构
```
# 《剧名》亮点梳理

## 一、爽点梳理
[列表形式，按时间顺序]

## 二、反转梳理
[列表形式，按时间顺序]

## 三、打脸梳理
[列表形式，按时间顺序]

## 四、亮点总结
[整体分析亮点的分布和效果]
```

### 单个条目格式
```
### 第X集 - [爽点/反转/打脸名称]
- **类型**：[具体类型]
- **情节概述**：[简要描述情节]
- **亮点分析**：[分析为什么这是亮点，带来什么效果]
```

## 注意事项
1. 基于定稿的分集剧情进行梳理，不要添加原创内容
2. 要客观准确地识别和分类
3. 分析要深入，说明亮点为什么有效
4. 注意亮点的密度和分布，是否均匀
5. 总结时可以提出优化建议（如果有）
