# 步骤四：分集剧本

## 任务目标
将定稿的分集大纲，改编成符合AI仿真人竖屏短剧叙事节奏和表现力的AI仿真人短剧的剧本

## 分拆规则

### 一、基础要求

#### 第1集特殊要求（强制执行）
**开头黄金30秒原则**：第1集的前3-4组分镜（约30-45秒）必须是全剧最精彩的内容！

**必须选择以下类型之一作为开篇**：
1. **最精彩设定展示**：全剧最吸引人、最有创新性的核心设定
   - 如：重生瞬间、身份揭露、超能力展示等

2. **最悬念场景**：全剧最让人好奇、最想知道答案的悬念场景
   - 如："你知道你明天会死吗？"、神秘物品、未知威胁等

3. **最强烈情绪**：全剧情绪最饱满、最能引起共鸣的场景
   - 如：被背叛的崩溃、死亡边缘的绝望、大快人心的打脸等

4. **最激烈冲突**：全剧矛盾最尖锐、对抗最激烈的场景
   - 如：婚礼现场撕逼、当众揭穿阴谋、生死对决等

**开篇后处理**：
- 精彩开篇结束后，用"时间回到X天前"、"这一切，还要从X说起"等方式，按线性顺序展开剧情
- 开篇与后续剧情的衔接要自然流畅

#### 时长适配
- 每集剧本对应**10-12组分镜**
- 每组分镜**6-15秒**
- 单集总时长控制在**1-2分钟**

#### 集数匹配
- 严格按照上面的分集剧情内容划分
- 不要增减集数
- 每集内容要与分集剧情对应

#### 主线连贯
每集需包含：
1. **前情回顾**：简要回顾上一集关键内容（第1集无需）
2. **本集剧情**：完整展现本集内容，合理补充细节以填充分镜
3. **结尾勾子**：保留或强化原文结尾勾子

#### 内容扩充策略
当分集剧情内容不足以填充10-12组分镜时：
1. **扩展人物反应**：将"主角很生气"扩展为"主角瞳孔收缩→拳头握紧→深呼吸平复→眼神变冷"
2. **增加对话层次**：将简单对话拆分为问答、确认、反问等多个回合
3. **添加动作序列**：增加人物的连贯动作，如从座位站起→走到窗前→望着窗外→缓缓开口
4. **插入环境细节**：通过环境描写增加氛围感，占用适当时长
5. **细化情绪变化**：展示情绪从产生到升级的完整过程

### 二、尊重原著与合理扩充

- **主线尊重**：尊重已确定好的分集剧情核心主线和关键情节
- **细节扩充**：可以在不影响主线的前提下，合理补充细节情节，确保单集内容足够填充10-12组分镜
- **人物性格一致**：保持人物性格一致
- **剧情逻辑一致**：保持剧情逻辑一致

#### 可补充的细节类型
1. **人物反应细节**：增加人物听到关键信息后的表情变化、眼神动作等
2. **环境互动细节**：增加人物与环境的互动，如拿起水杯、踱步、手势等
3. **对话缓冲细节**：在关键对话之间增加短暂的反应、停顿或确认
4. **铺垫细节**：增加一些细节为后续情节做微妙铺垫
5. **情绪递进细节**：将单一情绪拆分为递进的层次，展示情绪变化过程

### 三、内容结构

#### 按分镜组划分
- 每组分镜编排好序号
- 分镜组下编排内容
- 无需标注景别和运镜信息（seedance2.0模型会自动处理）

### 四、对白台词

#### 丰富对白
- 尽量以角色对白方式表现原文陈述性内容
- 不采用旁白方式
- 内心戏可转化为角色独白台词或蒙太奇画面

#### 台词精炼
- 角色台词简短有力
- 符合人设
- 避免冗长对话
- 竖屏观看，台词不宜过长

### 五、视听表现

- 按影视化的视听方式表现剧情内容
- 剧本描述以"动作+对话+情绪"为主
- 可包含环境描写、人物状态描写
- 画面感要强

## 分镜编写要点

### 分镜内容要素
每个分镜应包含：
1. **场景描述**：在哪里发生
2. **人物状态**：人物的穿着、表情、情绪
3. **动作**：人物在做什么
4. **对话**：人物说什么（如果有）
5. **氛围**：整体感觉和氛围

### 内容扩充技巧

#### 技巧1：拆解复杂情绪为多个分镜
```
原文：主角听到消息后很震惊

扩充为3个分镜：
分镜1：主角正在喝水，听到消息的瞬间愣住
分镜2：水杯从手中滑落，水洒在地上
分镜3：主角缓缓抬头，眼神中充满震惊和难以置信
```

#### 技巧2：增加对话回合
```
原文：主角质问反派，反派承认

扩充为多个分镜：
分镜1：主角眼神冰冷地看着反派："是你做的？"
分镜2：反派挑眉冷笑："是又怎样？"
分镜3：主角逼近一步："为什么？"
分镜4：反派摊摊手："谁让你挡路了"
分镜5：主角拳头握紧："你会后悔的"
```

#### 技巧3：增加环境和动作细节
```
原文：主角走进房间

扩充为2-3个分镜：
分镜1：主角站在门口，深吸一口气，手放在门把手上
分镜2：推开门，房间内的景象映入眼帘（环境描写）
分镜3：主角缓步走进，目光扫视四周
```

#### 技巧4：增加反应分镜
在关键情节后，增加其他人物的反应分镜
```
关键情节分镜后，增加：
分镜X：配角A震惊地捂住嘴
分镜X+1：配角B脸色发白，连连后退
分镜X+2：反派露出得意的笑容
```

### 分镜时长控制
- **短分镜（6-8秒）**：快速反应、简单对话、瞬间表情
- **中长分镜（9-12秒）**：完整对话、连贯动作、情绪展示
- **长分镜（13-15秒）**：复杂动作、环境展示、情绪递进

建议每集包含：6-7个中长分镜 + 3-5个短分镜 = 10-12分镜

### 竖屏适配
- 考虑竖屏观看特点
- 重点突出人物上半身和面部表情
- 避免过多的大场面
- 对话要适合近距离观看

### AI仿真人适配
- 表情和动作要适合AI仿真人表现
- 避免过于复杂的动作
- 情绪表达要清晰
- 对话口型要自然

## 第1集开篇示例

### 示例1：重生复仇类开篇（最激烈冲突）

**开篇3-4组分镜（黄金30秒）**：
```
分镜1：豪华婚礼现场，宾客满堂，新娘穿着婚纱却满脸是血
     新娘（林晚）指着新郎和伴娘："陈浩宇、张美美，你们这对狗男女！"
     陈浩宇冷笑："林晚，你太碍事了"

分镜2：张美美挽着陈浩宇的胳膊，得意地展示手上的钻戒
     张美美："姐姐，你的一切现在都是我的了，包括浩宇哥"
     林晚气得浑身发抖

分镜3：陈浩宇拿出一份文件，摔在林晚面前
     陈浩宇："这是你签字的股权转让书，林家的公司现在是我的了"
     林晚难以置信地摇头："不可能！我什么时候签的？"

分镜4：林晚被推倒在地，陈浩宇和张美美转身离去
     林晚绝望地大喊："我做鬼也不会放过你们！"
     画面变黑

【时间回到一个月前】
```

### 示例2：甜宠打脸类开篇（最悬念场景）

**开篇3-4组分镜（黄金30秒）**：
```
分镜1：高级餐厅，穿着朴素的女孩坐在对面
     男孩嫌弃地把一张支票推过去："这是50万，离开我儿子"
     女孩平静地看着支票

分镜2：女孩抬起头，眼神锐利
     女孩："阿姨，您确定要给我50万？"
     男孩母亲："怎么？嫌少？"

分镜3：这时，餐厅门打开，一群黑衣保镖走进来
     保镖们整齐地鞠躬："大小姐，总裁请您回家！"
     男孩母亲傻眼

分镜4：女孩站起身，优雅地擦了擦嘴
     女孩："阿姨，您的50万还是留着自己花吧"
     女孩转身离去，留下震惊的众人

【这一切，还要从三天前说起】
```

### 示例3：悬疑推理类开篇（最精彩设定）

**开篇3-4组分镜（黄金30秒）**：
```
分镜1：凶案现场，警察正在勘查
     死者倒在地上，旁边放着一个诡异的面具
     老刑警皱眉："又是这个面具...第三起了"

分镜2：一个年轻女孩走进现场，所有人都看着她
     女孩："我能看看尸体吗？"
     老刑警："你是谁？这里不是随便能进的"

分镜3：女孩闭上眼睛，手悬在尸体上方
     女孩（喃喃自语）："他是左撇子...身高185...穿黑色皮鞋..."
     老刑警震惊："你怎么知道？"

分镜4：女孩睁开眼睛，眼神坚定
     女孩："我能看到死者最后看到的画面"
     画面闪现一个神秘的黑影

【这个女孩，有一个不为人知的秘密】
```

## 注意事项
1. **第1集开篇强制执行**：必须选择最精彩/悬念/情绪/冲突的内容放在前3-4分镜
2. 每集严格控制在**10-12组分镜**，确保内容充足
3. 每组分镜时长**6-15秒**，单集总时长**1-2分钟**
4. **可以合理补充细节情节**：为了填充分镜，可以增加人物反应、动作序列、对话层次等细节，但不改变主线剧情
5. 扩充原则：细节要服务于人物塑造和情绪表达，不能为了凑时长而注水
6. 尽量用对话代替旁白和陈述
7. 台词要简短有力，符合人设
8. 画面感要强，适合影视化表现
9. 保持人物性格一致性
10. 第1集无需前情回顾
11. 保留并强化每集的结尾勾子
12. 时长检查：创作完成后检查单集时长是否在1-2分钟范围内
