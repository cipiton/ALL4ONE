## 概览
本文档定义小说改编为AI解说剧的评估规则框架，用于判断内容的适配度。用户可根据实际需求替换或补充具体规则。

## 评估维度

### 1. 情节完整性
- **完整故事线**:具备开端、发展、高潮、结局
- **逻辑连贯性**:情节发展合理，无明显逻辑漏洞
- **冲突设置**:有明确的冲突和矛盾
- **节奏把控**:情节张弛有度，适合1-2分钟/集的节奏

### 2. 人物鲜明度
- **主角形象**:主角特征清晰，有记忆点
- **人物关系**:角色关系网络合理
- **行为逻辑**:人物行为符合性格设定

### 3. 主题表达
- **核心思想**:有明确的主题或价值观
- **情感共鸣**:能引发观众情感共鸣
- **时代相关性**:内容与当下观众需求相关

### 4. 叙事适配性
- **视听化潜力**:内容易于通过画面和声音呈现
- **对话密度**:对话量适中，适合AI解说
- **场景转换**:场景数量合理，便于制作

### 5. 内容合规性
- **价值观导向**:符合主流价值观
- **版权清晰**:无版权争议
- **敏感内容**:无违规敏感内容

## 评分标准

每个维度采用10分制，总分50分：

| 分数段 | 适配度 | 建议 |
|--------|--------|------|
| 45-50 | 极高适配 | 强烈推荐改编 |
| 35-44 | 高度适配 | 推荐改编，微调优化 |
| 25-34 | 中度适配 | 需要较大调整 |
| 15-24 | 低度适配 | 不建议改编，需重大修改 |
| 0-14 | 不适配 | 不推荐改编 |

## 常见问题

### 问题1:情节过于复杂
- **表现**:支线过多，主线不清晰
- **影响**:难以在短集数内讲清楚
- **建议**:精简支线，聚焦主线

### 问题2:人物过多
- **表现**:角色超过10个且戏份均衡
- **影响**:观众难以记住
- **建议**:合并角色，突出主角

### 问题3:缺乏冲突
- **表现**:故事平铺直叙，没有矛盾
- **影响**:缺乏吸引力
- **建议**:增加外部冲突或内心冲突

### 问题4:主题模糊
- **表现**:核心思想不明确
- **影响**:观众难以产生共鸣
- **建议**:明确核心价值观

## 优化建议

### 增强适配度
1. **精简内容**:删减与主线无关的情节
2. **强化冲突**:在关键节点制造矛盾
3. **突出主题**:通过对话和情节明确核心价值观
4. **优化节奏**:控制每集信息量，确保流畅

### 适合改编的特征
1. **强情感**:能引发强烈情感共鸣
2. **强悬念**:有吸引人的悬念设置
3. **强记忆点**:有独特的设定或反转
4. **强共鸣**:反映现实生活中的普遍体验
